# Rakhine (rki) AI Lexicon — Full Package

This repository contains a full-stack, community-friendly, AI-ready lexicon for the Rakhine (rki) language.

Components:
- `schema/` — JSON Schema for entries
- `lexicon/` — per-word JSON entries
- `corpus/` — community submissions and cleaned corpora
- `audio/` — recordings + metadata
- `scripts/` — validation, dataset build, Meili indexing, training helpers
- `backend/` — FastAPI example
- `frontend/` — Next.js prototype
- `docs/` — guides
- `.github/workflows/` — CI for validation
